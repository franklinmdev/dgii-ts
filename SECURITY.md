# Security policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

If you discover a security vulnerability in dgii-ts, please report it
through [GitHub Security Advisories](https://github.com/franklinmdev/dgii-ts/security/advisories/new).

**Do not** open a public issue for security vulnerabilities.

### Scope

- Validators (RNC, cedula, NCF, e-NCF)
- SOAP client
- Bulk downloader/parser

### Response timeline

- **48 hours**: acknowledgment of your report
- **90 days**: coordinated disclosure deadline
